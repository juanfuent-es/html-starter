3. Variables CSS

Introducción a las Variables CSS

Variables CSS permiten definir valores reutilizables en una hoja de estilos.

Sintaxis

:root { --color_principal: #ff0000; --tamaño_fuente: 16px; }

Beneficios

  • Mantenibilidad del código
  • Reutilización de valores
  • Facilidad para hacer cambios

4. Temas en CSS

Definición de Temas

Un tema es un conjunto de estilos consistentes que definen la apariencia visual de una aplicación.

Componentes de un tema

  • Paletas de colores
  • Tamaño de fuente
  • Espaciado
  • Tipografía
  • Elementos reutilizables

5. Uso de Variables y Temas

Ejemplo práctico

:root { --color_fondo: #ffffff; --color_texto: #333333; --color_acento: #007bff; } body { background-color: var(--color_fondo); color: var(--color_texto); }

Aplicación en un tema

  • Definición de variables temáticas
  • Creación de estilos consistentes
  • Reutilización de elementos
