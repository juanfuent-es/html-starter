Programación Orientada a Objetos (POO)

Paradigma de programación que organiza el código en torno a objetos, que representan entidades del mundo real con propiedades y comportamientos.

Temario

  1. Conceptos Básicos
  2. Objetos
  3. Clases
  4. Herencia
  5. Polimorfismo
  6. Encapsulación

1. Conceptos Básicos

La POO se basa en cuatro pilares fundamentales: objetos, clases, herencia, polimorfismo y encapsulación. Estos conceptos permiten modelar entidades del mundo real en el código.

2. Objetos

Un objeto es una entidad que agrupa datos (propiedades) y funciones (métodos). Representa una instancia de una clase.

        
        // Ejemplo de objeto
        const persona = {
            nombre: "Juan",
            edad: 30,
            saludar: function() {
                return `Hola, mi nombre es ${this.nombre}.`;
            }
        };
        
        console.log(persona.saludar()); // Output: Hola, mi nombre es Juan.

3. Clases

Una clase es una plantilla para crear objetos. Define las propiedades y métodos que tendrán las instancias de la clase.

        
        // Ejemplo de clase
        class Persona {
            constructor(nombre, edad) {
                this.nombre = nombre;
                this.edad = edad;
            }
        
            saludar() {
                return `Hola, mi nombre es ${this.nombre} y tengo ${this.edad} años.`;
            }
        }
        
        const persona1 = new Persona("Juan", 30);
        console.log(persona1.saludar()); // Output: Hola, mi nombre es Juan y tengo 30 años.

4. Herencia

La herencia permite que una clase (clase hija) herede las propiedades y métodos de otra clase (clase padre).

        
        // Ejemplo de herencia
        class Animal {
            constructor(nombre) {
                this.nombre = nombre;
            }
        
            hacerSonido() {
                return `${this.nombre} hace un sonido.`;
            }
        }
        
        class Perro extends Animal {
            hacerSonido() {
                return `${this.nombre} ladra.`;
            }
        }
        
        const perro = new Perro("Firulais");
        console.log(perro.hacerSonido()); // Output: Firulais ladra.

5. Polimorfismo

El polimorfismo permite que un método se comporte de manera diferente según el objeto que lo invoque.

        
        // Ejemplo de polimorfismo
        class Animal {
            hacerSonido() {
                return "Este animal hace un sonido.";
            }
        }
        
        class Gato extends Animal {
            hacerSonido() {
                return "El gato maúlla.";
            }
        }
        
        const animal = new Animal();
        const gato = new Gato();
        
        console.log(animal.hacerSonido()); // Output: Este animal hace un sonido.
        console.log(gato.hacerSonido());   // Output: El gato maúlla.

6. Encapsulación

La encapsulación consiste en restringir el acceso directo a las propiedades de un objeto, permitiendo su manipulación solo a través de métodos.

        
        // Ejemplo de encapsulación
        class CuentaBancaria {
            #saldo; // Propiedad privada
        
            constructor(saldoInicial) {
                this.#saldo = saldoInicial;
            }
        
            depositar(cantidad) {
                this.#saldo += cantidad;
            }
        
            obtenerSaldo() {
                return this.#saldo;
            }
        }
        
        const cuenta = new CuentaBancaria(1000);
        cuenta.depositar(500);
        console.log(cuenta.obtenerSaldo()); // Output: 1500

Objetos en JavaScript

Un objeto es una estructura que agrupa datos (propiedades) y funciones (métodos) relacionados. Las propiedades son pares clave-valor, y los métodos son funciones asociadas al objeto.

Estructura de un Objeto

        
        const objeto = {
            clave1: valor1,
            clave2: valor2,
            metodo: function() {
                // Código del método
            }
        };

Cómo Funciona un Objeto

  • Propiedades: Datos almacenados en el objeto.
  • Métodos: Funciones asociadas al objeto que pueden manipular sus propiedades.
  • Acceso: Se puede acceder a las propiedades y métodos usando la notación de punto (objeto.propiedad) o de corchetes (objeto["propiedad"]).

Ejemplo Básico

        
        const persona = {
            nombre: "Juan",
            edad: 30,
            saludar: function() {
                return `Hola, mi nombre es ${this.nombre} y tengo ${this.edad} años.`;
            }
        };
        
        console.log(persona.nombre); // Output: Juan
        console.log(persona.saludar()); // Output: Hola, mi nombre es Juan y tengo 30 años.

Creación de Objetos

  • Objeto Literal: Se crea directamente usando llaves {}.
  • Objeto con Constructor: Se utiliza una función constructora para crear múltiples instancias.
  • Clases: Introducidas en ES6, son una forma moderna de crear objetos.

Ejemplo: Objeto Literal

        
        const coche = {
            marca: "Toyota",
            modelo: "Corolla",
            año: 2022
        };

Manipulación de Objetos

  • Agregar Propiedades: Se pueden agregar nuevas propiedades dinámicamente.
  • Eliminar Propiedades: Se utiliza la palabra clave delete.
  • Iterar sobre un Objeto: Se puede usar for...in para recorrer las propiedades.

Ejemplo: Agregar y Eliminar Propiedades

        
        const persona = {};
        persona.nombre = "Juan";
        persona.edad = 30;
        console.log(persona); // Output: { nombre: "Juan", edad: 30 }
        
        delete persona.edad;
        console.log(persona); // Output: { nombre: "Juan" }

Ejemplo: Iterar sobre un Objeto

        
        const persona = { nombre: "Juan", edad: 30 };
        for (let clave in persona) {
            console.log(`${clave}: ${persona[clave]}`);
        }
        // Output:
        // nombre: Juan
        // edad: 30
